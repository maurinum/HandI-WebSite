Comment ajouter vos vidéos YouTube ?

1

Obtenez une clé API YouTube

Rendez-vous sur Google Cloud Console et créez une clé API pour l'API YouTube Data v3

2

Configurez le service

Éditez le fichier src/app/services/youtube.ts et ajoutez votre clé API et Channel ID

3

Synchronisation automatique

Vos vidéos YouTube seront automatiquement récupérées et affichées sur cette page
